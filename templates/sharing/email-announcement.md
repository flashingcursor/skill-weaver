## How to install

### For claude.ai:
1. **Download**: https://github.com/flashingcursor/skill-weaver/releases/latest
2. **Upload**: Settings → Capabilities → Skills → Upload skill
3. **Enable**: Toggle the skill ON

### For Claude Code:
```bash
/plugin marketplace add flashingcursor/skill-weaver-marketplace
/plugin install skill-creator
```

Or see [installation guide](https://github.com/flashingcursor/skill-weaver/blob/master/INSTALLATION.md) for other methods.
